# 上虞市驾校官网 — 设计文档

> 日期：2026-06-23 | 状态：待用户审阅

## 项目概述

为上虞市驾校（位于绍兴市上虞区）开发官方展示网站。目标用户是想学车的学生及其家长，核心诉求是**建立信任、展示实力、获取报名线索**。

## 技术方案

**静态网站 + 第三方表单服务**

- 纯静态 HTML/CSS/JS 构建，无需服务器
- 在线报名使用第三方表单服务（如金数据/腾讯问卷）收集信息
- 部署到静态托管平台（GitHub Pages / Netlify / 阿里云 OSS 等）

## 视觉风格

| 要素 | 选择 |
|------|------|
| **主题** | 大气高端风 — 深蓝金 |

## 页面结构（单页滚动 + 锚点导航）

页面从上到下共 8 个区块：

### 1. 导航栏（固定顶部）
- 驾校名称 + Logo（左侧）
- 锚点链接：首页 / 驾校优势 / 课程价格 / 教练团队 / 学员评价 / 联系我们
- "在线报名"按钮（右侧，金色实心）
- 滚动时保持固定，半透明毛玻璃效果

### 2. 首屏 Hero
- 大标题：「学车，从上虞市驾校开始」
- 四句卖点横向排列：98% 通过率 · 透明收费 · 耐心教练 · 45 天快速拿证
- 两个 CTA 按钮：立即报名（金色实心）/ 了解课程（金色边框）
- 底部有向下滚动提示箭头
- 背景深蓝渐变 + 金色光晕

### 3. 驾校优势（四大理由）
- 标题：「选择上虞市驾校的四大理由」
- 四列卡片网格，每张卡片包含图标 + 数字/关键词 + 说明
  - 🎯 98% 通过率 — 科目二三一次过，远超本地平均水平
  - 💰 透明收费 — 一口价全包，合同写清，绝无隐形收费
  - 👨‍🏫 耐心教学 — 5 年以上教龄，不骂人、不敷衍、手把手教
  - ⏱️ 45 天拿证 — 合理安排练车时间，最快 45 天拿到驾照

### 4. 课程价格（仅 VIP 班）
- 标题：「课程与价格」
- **仅展示一个 C1/C2 VIP 班**：¥5,800
- 权益列表：不限时练车、周末可练、优先预约考试、接送服务、一对一教学、免费补考
- 一个醒目的金色 CTA 按钮：「立即报名」

### 5. 教练团队
- 标题：「经验丰富的教练团队」
- 2 位教练卡片（照片 + 姓名 + 教龄 + 擅长领域 + 个人介绍）
- 罗教练（男）— 从业 30 年，资深教练，C1/C2 都擅长
- 沈教练（女）— 从业 15 年，亲和力强，C1/C2 都擅长
- 照片由用户后续上传，先使用占位图

### 6. 学员评价
- 标题：「听听学员们怎么说」
- 3 条评价卡片，左侧金色竖线装饰
- 包含星级、评语、学员姓名和年份

### 7. 报名表单
- 标题：「现在报名，开启学车之旅」
- 字段：姓名、手机号、意向课程（下拉选择）、留言（选填）
- 提交按钮 → 数据发送到第三方表单服务
- 提交后显示成功提示

### 8. 页脚
- 三栏布局：联系方式 / 快速导航 / 社交媒体
- 版权信息和 ICP 备案号

## 响应式设计

- 桌面端（>768px）：多列网格布局，完整导航栏
- 移动端（≤768px）：单列堆叠，汉堡菜单，触摸友好的按钮尺寸

## 数据流

```
用户填写表单 → 提交 → 第三方表单服务（金数据等）→ 教练收到通知（微信/短信）
```
